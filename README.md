# GeekProject
极客家，想做成一个结束类的新闻客户端，同时兼顾日常查询的一些使用。

## 模块
### GankIO
代码家的API接口提供的内容

### 程序猿老黄历
程序猿老黄历实现

### 高德地图
定位功能以及高德地图聚合实现

### 天气预报
使用高德地图sdk实现的天气查询以及实时天气

## Features 特性
* 遵循Material Design设计风格。
* Rxjava+Retrofit2网络请求，并进行封装错误可以进行统一处理。
* 使用SwipeRefreshLayout+RecycleView进行下拉刷新
* Glide做图片的处理和加载
* 高德地图使用的封装
